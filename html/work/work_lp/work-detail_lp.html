LP制作物

WEB SITE / アウトドア LP制作 2025年9月 1日間程度 Photoshop

課題・背景

ペルソナ（ターゲット像） 年齢層：20〜30代
ライフスタイル：都市部で働いているが、休日は自然に触れてリフレッシュしたい
悩み：日常が忙しく、リフレッシュする時間が少ない
アウトドアに挑戦したいけど、何から始めればいいか分からない
重くてかさばるギアは使いづらい
価値観：自然と調和するシンプルなデザインを好む
無理のない範囲でアウトドアを楽しみたい
インスタやSNSで「雰囲気の良い体験」をシェアしたい

アプローチ（工夫した点）

  • 解決策（このLPが提供する価値）

    軽量で持ち運びやすいリュック → 初心者でもストレスなく使える
    耐久性のある焚火台 → 長く使えてコスパが良い
    自然に溶け込むカラーデザイン → SNS映えしつつ落ち着いた雰囲気を演出
    週末に気軽に使えるギア → 本格的な登山ではなく、ゆるキャンやデイキャンプにも最適

  • 工夫した点（デザイン・ライティングの意図）

    色味の工夫
    全体を「深緑・ベージュ・ウッド調」で統一 → 自然に溶け込む安心感を演出
    コピーライティング
    「自然と、もっと近く。」など短く感情に訴えるフレーズを採用
    「週末」「やさしい時間」など、ターゲットが共感しやすいキーワードを使用
    ビジュアル表現
    テントや焚火を囲むシーン → 週末に手軽に楽しむアウトドアをイメージできる
    森や夕暮れの写真 → リラックス感と非日常感を強調
    ストーリーテリング型の構成
    ファーストビュー → ブランドコンセプト → プロダクト紹介 → ライフスタイル提案 → クロージング
    物を売るだけでなく、「体験の価値」を訴求

Web Site 制作物

CoreFlow(友達のポートフォリオサイト) 2025年9月〜10月 Photoshop/Corsor

課題・背景

地域に根ざした新規コーヒーショップのLP制作。
大手チェーンとの差別化が難しく、
まずは「温かみ」や「こだわり」を伝えて認知を広げることが課題でした。

アプローチ（工夫した点）

  • ブランドカラーに合わせた温かみのあるデザインで個性を表現
  • 写真を大きく配置し、焙煎やハンドドリップの「こだわり」を直感的に伝達
  • 「想い → 商品紹介 → 来店導線」の流れを意識したストーリー設計
  • 余白を多く使い、短く簡潔なテキストで読みやすさを確保
  • 店舗情報や地図を最後に配置し、来店へのスムーズな導線を設計

作品ビジュアル

# Web Designer Portfolio

顧客に寄り添うWebデザイナーのポートフォリオサイト

## デプロイ方法

### Netlifyでのデプロイ（推奨）

1. [Netlify](https://app.netlify.com)にアクセス
2. 「Add new site」→「Deploy manually」を選択
3. このフォルダ全体をドラッグ&ドロップ
4. デプロイ完了！

## プロジェクト構成

- `html/` - HTMLファイル
- `css/` - スタイルシート
- `images/` - 画像ファイル

## 制作実績

- SNS投稿デザイン
- LP制作
- Webサイト制作
- ロゴデザイン
